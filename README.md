# Wordparrot Plugin Starter

Featuring:

- Webpack 5
- Typescript
- Scss
- Ts-jest
- Eslint

## Getting started

Make sure to have NodeJs already installed.

### Install all dependencies

```bash
npm install
```

### Start developing and serve your app

```bash
npm start
```

### Build your application

```bash
npm run build
```

### Run unit tests

```bash
npm run test
```

### Run coverage

```bash
npm run coverage
```
